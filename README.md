## Project Setup

### Installation

`composer install`

`cp .env.example .env`

`php artisan key:generate`

`php artisan migrate`

### Fetch tasks from providers

`php artisan tasks:fetch provider1`

`php artisan tasks:fetch provider1`

### Serve project

`php artisan serve`

Finally, visit the homepage at:

`http://127.0.0.1:8000`
